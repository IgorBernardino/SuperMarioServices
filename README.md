## 🚀 Projeto

### 🎮 Mario Services

**Stack:** HTML e CSS

Site temático com identidade visual inspirada na franquia Super Mario. Apresenta serviços fictícios de forma criativa e lúdica, com navegação entre duas páginas — a inicial e a de serviços — cada uma com seu próprio arquivo de estilos.

**Páginas:**
- `index.html` — Página principal / landing page
- `nossos-servicos.html` — Listagem de serviços oferecidos

**Conceitos aplicados:**
- Estruturação semântica com HTML
- Estilização temática e criativa com CSS
- Separação de estilos por página (`style.css` e `services.css`)
- Navegação entre múltiplas páginas em site estático
- Organização de assets (imagens e ícones)

## ▶️ Como Usar

Projeto estático, sem dependências ou build necessário.

```bash
# Clone o repositório
git clone https://github.com/IgorBernardino/SuperMarioServices.git

# Acesse a pasta do projeto
cd SuperMarioServices/mario

# Abra o index.html no navegador
# Ou use a extensão Live Server no VS Code
```
